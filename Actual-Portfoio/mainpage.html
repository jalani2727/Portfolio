#2 : responsive header

The basic idea is to change the display property for the #head div (right menu) from flex to table and adjust width and height. When the screen is under 640px the main menu sticks to the top, by changing the <li> elements from default to inlint, they distribute correctly. 

      @media screen and (max-width: 640px)
      {
        #head {
          height: 125px;
          display: table;
          width: 100%;
        }
        #head ul li {
          display: inline;
        }
        #content  {
          margin-left: 0;
          margin-top: 125px;
        }
      }
